📋 ToDo List Application with Spring Boot
A simple and intuitive ToDo List web application, built with Spring Boot and Thymeleaf, where each user can manage their personal tasks after logging in. The app allows users to create, view, update, and delete their own tasks in a secure and user-friendly environment.

🚀 Features
✅ User registration and login

🔒 Session-based authentication

📝 Create, read, update and delete tasks (CRUD)

🙋‍♂️ Each user sees only their own tasks

📅 Tasks have title, description, due date, and status

📄 Frontend with Thymeleaf templates

🎯 MVC architecture with Spring Boot

🛠 Technologies Used
Java 17+

Spring Boot

Spring MVC

Spring Security

Thymeleaf

Spring Data JPA

MySQL

Lombok

Bootstrap 5
